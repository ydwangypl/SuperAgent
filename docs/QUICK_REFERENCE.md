## 🎯 核心概念

### SuperAgent 是什么?

**Python 库/模块**,不是独立工具!

```
您 (在 Claude Code 中)
    ↓
对我说: "使用 SA 开发..."
    ↓
我执行 Python 代码:
  from orchestration import Orchestrator
  result = await orchestrator.execute_plan(plan)
    ↓
SA 返回需求文档
    ↓
我根据需求生成代码
```

### 三大核心功能

1. **智能规划** - 自动分解任务
2. **记忆系统** - 防止重复错误
3. **代码审查** - 提升代码质量

## ⚡ 高级用法

### 自定义配置

```
您: 使用 SA 开发...,配置: 最大并行任务5,启用审查

我执行:
  config = OrchestrationConfig(
      max_parallel_tasks=5,
      enable_review=True
  )
  orchestrator = Orchestrator(Path("."), config=config)
```

### 查看详细结果

```
您: 使用 SA 开发博客系统,显示详细步骤

我执行:
  - 显示每一步的计划
  - 显示每个 Agent 的返回结果
  - 显示需求文档内容
  - 显示代码审查报告
```

## 💡 最佳实践

### ✅ 推荐做法

1. **复杂项目用 SA**
   - "使用 SA 开发电商系统"

2. **遇到 Bug 先查记忆**
   - "使用 SA 查询记忆: 支付失败"

3. **定期代码审查**
   - "使用 SA 审查整个项目"

4. **重构前先规划**
   - "使用 SA 规划重构为微服务"

### ❌ 不推荐

1. ❌ 简单任务用 SA
   - 直接说: "创建一个 user.py 文件"

2. ❌ 忽略记忆建议
   - 可能重复历史错误

3. ❌ 跳过代码审查
   - 可能遗漏质量问题
